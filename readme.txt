Aby uruchomić projekt należy mieć zainstalowanego:

Node.js
Konfiguracja:

W konsoli należy przejść do folderu src i za pomocą komendy npm install zainstalować wszystkie zależności projektu. 
Gdy pobieranie dobiegania końca w terminalu wpisać komendę node app.js, która uruchomi aplikację. 
W konsoli powinien znaleźć się link do przeglądarki, który należy uruchomić lub wkleić http://localhost:5000/ .
